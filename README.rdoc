= Redmine Issue Status Colors

Adds extra css classes for the issue status


== Getting the plugin

A copy of the plugin can be downloaded from {Little Stream Software}[https://projects.littlestreamsoftware.com/projects/redmine-misc/files] or from {GitHub}[http://github.com/edavis10/redmine_issue_status_colors]


== Installation and Setup

1. Follow the Redmine plugin installation steps at: http://www.redmine.org/wiki/redmine/Plugins 
2. Configure any custom CSS you want to use (see below)
3. Restart your Redmine web servers (e.g. mongrel, thin, mod_rails)
